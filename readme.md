# Movie Recommendation System

## Description
This project is a simple movie recommendation system developed in Python. It recommends movies based on the user's preferred genre.

## Features
- Recommends movies by genre
- Displays a random top pick
- Allows multiple recommendations
- Easy-to-use command-line interface

## Technologies Used
- Python 3

## How to Run

```bash
python recommendation.py
```

## Genres Available
- Action
- Comedy
- Horror
- Romance
- Sci-Fi
